## 🎯 新增功能详解

### 1. DrawCircle - 绘制圆形

**用途**: 创建圆柱、球体等回转体

**参数**:
- `centerX` - 圆心X坐标（mm）
- `centerY` - 圆心Y坐标（mm）
- `radius` - 半径（mm）

**示例**:
```python
helper.DrawCircle(0, 0, 25)  # 绘制半径25mm的圆
```

**应用场景**:
- 圆柱体（圆形 + 拉伸）
- 管道（圆形 + 拉伸）
- 回转体（圆形 + 旋转）

### 2. DrawLine - 绘制直线

**用途**: 创建复杂草图轮廓

**参数**:
- `x1, y1` - 起点坐标
- `x2, y2` - 终点坐标

**示例**:
```python
helper.DrawLine(0, 0, 100, 0)  # 绘制100mm水平线
```

**应用场景**:
- 多边形轮廓
- 复杂截面
- 草图构造

### 3. CreateRevolution - 创建旋转特征

**用途**: 创建圆柱、球体、圆锥等回转体

**参数**:
- `angle` - 旋转角度（弧度）
  - 360度 = 2π ≈ 6.28318
  - 180度 = π ≈ 3.14159

**示例**:
```python
helper.CreateRevolution(6.28318)  # 旋转360度
```

**应用场景**:
- 圆柱体（矩形 + 旋转）
- 球体（半圆 + 旋转）
- 圆锥体（三角形 + 旋转）

### 4. CreateFillet - 创建圆角

**用途**: 边缘圆角处理

**参数**:
- `radius` - 圆角半径（mm）

**示例**:
```python
helper.CreateFillet(5.0)  # 5mm圆角
```

**应用场景**:
- 边缘平滑
- 倒圆处理
- 美观设计

### 5. CreateChamfer - 创建倒角

**用途**: 边缘倒角处理

**参数**:
- `distance` - 倒角距离（mm）
- `angle` - 倒角角度（度）

**示例**:
```python
helper.CreateChamfer(2.0, 45.0)  # 2mm距离，45度角
```

**应用场景**:
- 装配导向
- 边缘处理
- 工艺要求

## 💡 完整使用示例

### 示例1: 创建圆柱体（直径50mm，高30mm）

```python
import win32com.client

helper = win32com.client.Dispatch("SWHelper.SWHelperDynamic")

# 连接SolidWorks
helper.ConnectToSW()

# 创建零件
helper.CreatePart()

# 创建草图
helper.CreateSketch()

# 绘制半径25mm的圆（直径50mm）
helper.DrawCircle(0, 0, 25)

# 关闭草图
helper.CloseSketch()

# 选择草图
helper.SelectSketch("Sketch1")

# 拉伸30mm（或者使用旋转：CreateRevolution(6.28318)）
helper.CreateExtrusion(30)

print("圆柱体创建完成：直径50mm x 高30mm")
```

### 示例2: 创建带圆角的方块

```python
import win32com.client

helper = win32com.client.Dispatch("SWHelper.SWHelperDynamic")

helper.ConnectToSW()
helper.CreatePart()

# 创建100x100的方形草图
helper.CreateSketch()
helper.DrawRectangle(0, 0, 100, 100)
helper.CloseSketch()

# 拉伸50mm
helper.SelectSketch("Sketch1")
helper.CreateExtrusion(50)

# 添加5mm圆角
helper.CreateFillet(5.0)

print("带圆角的方块创建完成：100x100x50mm，5mm圆角")
```

### 示例3: 创建球体（半径25mm）

```python
import win32com.client
import math

helper = win32com.client.Dispatch("SWHelper.SWHelperDynamic")

helper.ConnectToSW()
helper.CreatePart()

# 创建草图
helper.CreateSketch()

# 绘制半圆（半径25mm）
helper.DrawCircle(0, 0, 25)
# 需要添加直线和修剪来创建半圆轮廓

# 旋转360度创建球体
helper.CreateRevolution(2 * math.pi)  # 360度

print("球体创建完成：半径25mm")
```
