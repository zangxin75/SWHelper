### 为什么所有方法都失败？

**核心发现**：
```
VBA测试成功的条件：
✓ 零件是手动创建的
✓ 文档在UI中完全初始化
✓ 用户交互触发的操作

C#/Python失败的原因：
✗ 零件是程序创建的
✗ 文档处于"半初始化"状态
✗ 无用户交互的自动化调用
```

**关键差异**：
- **手动创建**：文档完全初始化，所有属性可用
- **程序创建**：文档处于限制模式，某些API被禁用
