```bash

双击: 部署Agent.bat

# 3. 开始对话
👤 您: 创建一个 100x100x50mm 的方块
🤖 Agent: ✅ 设计完成！
```

### 方式2: Claude增强模式

```bash
# 1. 设置API密钥
set ANTHROPIC_API_KEY=your-key

# 2. 启动Claude版本
双击: 启动Claude增强Agent.bat

# 3. 享受智能设计
👤 您: 设计一个轴承座
🤖 Claude: 🧠 理解需求 → 提供方案 → 执行设计 → 优化建议
```

### 方式3: 集成开发

```python
# 在您的代码中调用
from agent_coordinator import SolidWorksAgentCoordinator

async def my_design_task():
    agent = SolidWorksAgentCoordinator()
    result = await agent.process_design_request(
        "创建自定义零件"
    )
    return result
```
