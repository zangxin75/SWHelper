## 🎯 文档分类体系

### 一级分类（按用途）

| 分类 | 目录 | 用途 |
|------|------|------|
| 📄 **文档** | `文档/` | 所有项目文档 |
| 💻 **代码** | `代码/` | 源代码文件 |
| 🎯 **项目** | `项目/` | 独立子项目 |
| 📦 **模型** | `模型文件/` | SolidWorks模型文件 |
| 🗂️ **配置** | `配置/` | 系统配置文件 |

### 二级分类（按类型）

#### 文档目录细分
```
文档/
├── 研究报告/     # 技术调研、分析报告
├── 使用指南/     # 用户手册、教程
└── 技术文档/     # API文档、架构设计
```

#### 代码目录细分
```
代码/
├── Python脚本/   # Python自动化脚本
├── VBA宏/        # SolidWorks宏
└── 测试代码/     # 测试和验证代码
```

#### 模型文件目录细分
```
模型文件/
├── 零件/         # 3D零件(.SLDPRT)
├── 装配体/       # 装配体(.SLDASM) [预留]
└── 工程图/       # 工程图(.SLDDRW) [预留]
```

## 📝 文档命名规范

### 1. 研究报告类

**格式**: `{主题}_研究报告.md`

**示例**:
- `SolidWorks_2026_AI_Agent_自动化研究报告.md`
- `Claude_Code_API集成研究报告.md`
- `性能基准测试研究报告.md`

**模板**:
```markdown
# {主题}研究报告

**报告日期**: YYYY-MM-DD
**研究主题**: {简短描述}
**作者**: {作者名称}

## 执行摘要
{1-2段总结核心发现}

## 研究背景
{背景介绍}

## 主要发现
{发现内容}

## 结论与建议
{结论和建议}

## 参考资料
{链接和引用}
```

### 2. 使用指南类

**格式**: `{功能}_快速启动指南.md` 或 `{功能}_使用手册.md`

**示例**:
- `AI_Agent_快速启动指南.md`
- `MCP服务器部署手册.md`
- `Python开发环境搭建指南.md`

**模板**:
```markdown
# {功能}使用指南

## 🚀 快速开始
{5分钟内可完成的步骤}

## 📋 详细说明
{详细的操作步骤}

## 💡 使用示例
{实际使用案例}

## ❓ 常见问题
{FAQ}

## 🔗 相关资源
{相关链接}
```

### 3. 技术文档类

**格式**: `{组件}_API文档.md` 或 `{组件}_技术文档.md`

**示例**:
- `MCP服务器API文档.md`
- `COM适配器技术文档.md`
- `Agent框架架构设计.md`

**模板**:
```markdown
# {组件}技术文档

## 概述
{组件简介}

## API参考
{接口说明}

## 数据模型
{数据结构}

## 配置选项
{配置说明}

## 故障排除
{问题诊断}
```

## 🗂️ 新文档添加流程

### 步骤1: 确定文档类型

```

研究分析类？
├─ 是 → 放入 `文档/研究报告/`
```

## 🔄 文档生命周期管理

### 创建阶段

1. **需求确认**: 确认文档需求和目标读者
2. **类型确定**: 按分类体系确定文档类型
3. **模板选择**: 选择合适的文档模板
4. **内容创建**: 按模板创建文档内容
5. **审核**: 检查内容和格式
6. **发布**: 放入对应目录，更新索引

### 维护阶段

1. **定期审查**: 每月检查文档的准确性
2. **版本控制**: 重大修改时更新版本号
3. **变更记录**: 在文档末尾记录修改历史
4. **废弃处理**: 不再需要的文档移入`archive/`子目录

## 🚀 快速命令参考

### 文档操作

```bash
# 查找所有文档
find 文档/ -type f -name "*.md"

# 查找最近7天修改的文档
find 文档/ -type f -name "*.md" -mtime -7

# 统计文档数量
find 文档/ -type f -name "*.md" | wc -l

# 创建新文档（带模板）
cp "模板/研究报告模板.md" "文档/研究报告/新主题_研究报告.md"
```

### 目录维护

```bash
# 检查目录结构
tree -L 2 -d

# 清理空目录
find . -type d -empty -delete

# 查看目录大小
du -sh 文档/* 代码/* 模型文件/*
```
